# Book Review Application

This is a full-stack application that allows users to view, add, edit, and delete book reviews.

## Features
- View all book reviews
- Add new reviews
- Edit existing reviews
- Delete reviews

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
